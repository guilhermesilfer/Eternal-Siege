# Eternal-Siege
Eternal Siege é um jogo de defesa de base feito para a matéria de Linguagem de Programação I

**VÍDEO TESTANDO: https://drive.google.com/file/d/1hCdsMtC-DHb0Gz9zXQqlXNIQWMxIO1rT/view?usp=sharing**

- **SFML 2.6**

- **Compilador C++** (g++)

- **make**


## Compilação
### make
```
make
```

## Execução
### make
```
make run
```
### ou
```
./bin/Eternal_Siege
```
